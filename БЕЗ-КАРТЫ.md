# Работа без карты (без Render)

Render сейчас **требует карту** даже для free. Обойтись можно так:

## 1. Сайт (уже работает)

https://kirikov1982evgenii-sketch.github.io/klub-znaniy-hart/

## 2. API и бот — PythonAnywhere (бесплатно, карта не нужна)

1. Регистрация: https://www.pythonanywhere.com/registration/register/beginner/  
2. **Web** → **Add a new web app** → Manual configuration → **Python 3.12**  
3. **Code** → загрузите файлы проекта (или `git clone` в Bash).  
4. **WSGI configuration file** → укажите:

```python
import sys
path = '/home/ВАШ_ЛОГИН/klub-znaniy-hart'  # путь к проекту
if path not in sys.path:
    sys.path.insert(0, path)
from wsgi import application
```

5. **Web** → **Environment variables** — скопируйте из `DEPLOY-HART-Render.txt` на рабочем столе.  
6. **Reload** веб-приложения.  
7. Ваш API: `https://ВАШ_ЛОГИН.pythonanywhere.com/api/health`  
8. В GitHub: **Settings → Secrets and variables → Actions → Variables** →  
   `MARKET_PAYMENT_API` = `https://ВАШ_ЛОГИН.pythonanywhere.com`  
9. Сделайте **push** в `main` или перезапустите workflow — сайт подхватит новый API.  
10. Webhook бота:

```bash
python scripts/set-telegram-webhook.py https://ВАШ_ЛОГИН.pythonanywhere.com
```

Клуб знаний HART → GitHub

Файлы готовы в этой папке. Нажмите кнопки по порядку:

1. Создать репозиторий github.com/new

  1. Имя репозитория: hart
  2. Без галочки «Add README» → Create repository
  3. На новой странице: ссылка uploading an existing file
  4. Откройте папку Клуб-знаний-HART-GitHub рядом на рабочем столе
  5. Ctrl+A → перетащите все файлы в браузер
  6. Commit changes

После загрузки в репозитории откройте файл DEPLOY.md — публикация сайта в интернет.

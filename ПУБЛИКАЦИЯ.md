## API оплат и бот — Render (один раз)

1. Откройте: https://dashboard.render.com/blueprint/new?repo=https%3A%2F%2Fgithub.com%2Fkirikov1982evgenii-sketch%2Fhart  
2. **Apply** → создадутся `hart-payments` и `hart-telegram-bot`.  
3. Вставьте переменные из `ДЕПЛОЙ-HART-переменные-Render.txt` на рабочем столе.  
4. Проверка: https://hart-payments.onrender.com/api/health → `{"ok":true}`

Или запустите: `powershell -File scripts\publish-render.ps1`

# Что получает покупатель (199 ₽)

## Входит в доступ HART

- Дорожная карта курса на **300–520+ часов** (структура модулей).
- Блоки-уроки: **видео RuTube** + конспект (пункты) + **задания на сайте** + итоговый тест.
- Личный кабинет: список купленных курсов, сохранение ответов в заданиях (в браузере).

## Не входит (важно для ожиданий)

- **Сертификат Coursera / NPTEL / вуза** — только у партнёра по его правилам.
- **Персональный репетитор** и проверка всех домашних HART преподавателем.
- **Уникальный видеокурс с диктором под каждый из 231 курсов** — используются обучающие ролики RuTube + текст программы HART.

## Частые вопросы

**Почему видео не про мой точный курс?**  
Ролик даёт методику и контекст (IT, MOOC, школа). Конкретика курса — в пунктах урока и заданиях; ссылка на партнёра — в справке «О продукте».

**Где проходить экзамен NPTEL / Coursera?**  
На сайте партнёра (`partnerUrl` в карточке курса). HART — навигатор и программа, не замена платформы.

**Не открывается видео**  
Откройте «На RuTube» под плеером. Нужен интернет; YouTube в РФ часто недоступен — приоритет RuTube.

**Пустой урок**  
Обновите страницу (Ctrl+F5). Если повторяется — напишите номер курса из URL (`n=`) и id курса.

## Поддержка

Перед ответом покупателю проверьте: оплата в кабинете, курс открыт с `learn=1` только для теста, для покупателя — после входа email + код PAY-.

# Агент фриланса (Cursor + @workmy1bot)

## Что делает (полный цикл)

1. **Ищет** заказы FL.ru + весь мир (RSS/API)
2. **Откликается** на подходящие (только под Cursor: код, боты, парсеры, API)
3. **Общается** в чатах FL: ТЗ, сроки, цена, предоплата 30%, реквизиты Т-Банк
4. **После оплаты** — генерирует код, запускает скрипты, шлёт архив заказчику в чат

Оплата: **Т-Банк**, карта в `курсор/fl-orders-bot/.env` → `PAYMENT_CARD`

## Запуск

Постоянный режим (каждые ~90 сек новые заказы + чаты):

```bat
start-freelance-agent.bat
```

Разово «всё сразу» (сессия FL → отклики → чаты → выполнение):

```bat
run-full-cycle.bat
```

В Telegram откройте **@workmy1bot** → `/start` → `/help`

## Команды в боте

| Команда | Действие |
|---------|----------|
| `/start` | Подключить чат |
| `/now` | Подборка заказов сейчас |
| `/world` | Заказы вне FL.ru |
| `/test` | Проверить RSS |
| `/pay` | Реквизиты Т-Банк для заказчика |
| `/status` | Статус и лиды |
| `/pause` / `/resume` | Пауза мониторинга |

## Важно

- **Не запускайте** `telegram_bot.py` (@uportbot, курсы HART) **одновременно** с агентом — разные боты, но один ПК = один polling на токен.
- Агент фриланса = **@workmy1bot** (`курсор/fl-orders-bot/.env`).
- Курсы HART = **@uportbot** (`telegram_bot.py`).
- FL.ru: для автоотклика нужна сессия браузера (Firefox/Yandex) — см. `fl-orders-bot/README.md`.

## Настройки

Переменные в `.env.local` (проект HART) или `курсор/fl-orders-bot/.env`:

- `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` — в `курсор/fl-orders-bot/.env`
- `MARKET_PAY_CARD`, `MARKET_PAY_BANK`, `MARKET_PAY_NAME` — оплата

Код агента: `курсор/fl-orders-bot/` + фильтр `cursor_agent.py`
